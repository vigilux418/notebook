# 数据库系统

!!! abstract "课程简介"

    **数据库系统** `Database Systems`,简称`DB`，是研究如何科学地**组织**、**存储**、**管理**和**维护大规模数据**的学科。
    
    课程主要内容分为两部分：
    
    1. 数据库应用
    
          - 数据库介绍和关系模型
          - `SQL` 的使用，包括基本概念、语法和高级的函数，比较复杂的嵌套
          - 数据库的表的设计
    
    2. 数据库管理系统的实现
    
          - 计算机存储设备和层级结构
          - 索引、查询处理、查询优化、事务与并发控制、故障恢复等
    
    **数据**是现代互联网的核心资产。掌握数据库对于企业级软件开发、金融数据处理以及NoSQL分布式数据库的研究至关重要。
