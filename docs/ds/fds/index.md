# 数据结构基础

!!! abstract "课程简介"

    **数据结构基础** `Fundamental Data Structures`， 简称 `FDS`，是计算机科学中关于如何高效组织和存储数据，以支持高效访问和修改的核心课程。
    
    课程内容包括：**栈、队列、树、堆、并查集、图**等**数据结构**，以及**最短路、搜索、网络流、排序、哈希**等**算法**。
    
    掌握数据结构是进行高效**算法设计**、优化程序性能的前提，是后续学习**操作系统**、**数据库系统**、**编译器**、**计算机网络**等几乎所有计算机专业课程的基础，是计算机专业学生的核心技能，也是编写任何高效软件的灵魂所在。
