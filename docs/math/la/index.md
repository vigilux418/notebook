# 线性代数

!!! abstract "课程简介"

    **线性代数** `Linear Algebra`是研究向量空间、线性映射及线性方程组的数学分支，是**现代计算科学的语言**。
    
    主要内容：**线性空间**、**线性映射与矩阵**、**行列式**、**线性方程组**以及**相似对角化**等。
    
    **线性代数**是**机器学习**和**深度学习**的核心（如张量运算），也是**计算机图形学**中实现坐标变换、投影成像的基础。在**大数据处理**和**量子计算**领域，线性代数是描述复杂系统状态转换的基本工具，也是这些领域的**核心语言**。
