# 概率论与数理统计

!!! abstract "课程简介"

    **概率论与数理统计** `Probability Theory and Mathematical Statistics`是研究随机现象统计规律性的数学学科。
    
    主要内容包括**概率论**和**数理统计**两个部分。
    
    1. 概率论：**随机变量及其分布**、**数字特征**、**大数定律**与**中心极限定理**
    2. 数理统计：**参数估计**、**假设检验**、**回归分析**与**方差分析**
    
    **概率论与数理统计**是**人工智能**中概率图模型、**贝叶斯网络**的理论源头。在**网络通信**的信噪比分析、**分布式系统**的容错评估以及**信息论**中，它也提供了量化随机现象的数学框架。它为计算机系统在不确定环境下的建模、推断与决策提供了科学依据。
