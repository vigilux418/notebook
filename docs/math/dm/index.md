# 离散数学及其应用

!!! abstract "课程简介"

    **离散数学及其应用** `Discrete Mathematics and Its Application`，简称`DM`，是计算机科学的理论基础和核心数学工具。
    
    主要内容：逻辑，集合，基本的算法，计数原理，生成函数，关系，图，树。
    
    离散数学在计算机科学中地位显赫，为数理逻辑、可计算性理论、算法设计与分析、数据结构、数据库理论、形式语言与自动机理论、密码学以及计算机网络等关键领域提供了**严谨的数学语言和分析框架**。它是理解计算本质、进行形式化验证和算法复杂度的**基石**。
