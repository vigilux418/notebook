# 软件工程

!!! abstract "摘要"

    **软件工程** `Software Engineering`关注于以工程化方法构建和维护高质量、实用且高效的软件系统。
    
    **软件工程**主要涵盖**软件开发生命周期 (SDLC)**、**设计模式**、**软件架构设计**（微服务、单体架构）、**软件测试与质量保障**、以及**敏捷开发方法论**。
    
    **软件工程**处于**最高维度的管理抽象**，旨在通过规范化的流程、方法、工具和管理来应对软件的复杂性、可变性和团队协作挑战。
    
    在软件开发日益规模化、复杂化的今天，软件工程是确保项目成功、控制成本与风险、保障软件可维护性与可靠性的关键。它是连接个体编程能力与产业级软件产品交付的桥梁。
