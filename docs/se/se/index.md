# 软件工程

!!! abstract "课程简介"

    **软件工程** `Software Engineering`是将系统化、规范化、可量化的方法应用于软件的开发、运行和维护的工程学科。
    
    主要内容：**软件开发生命周期 (SDLC)**、**敏捷开发 (Agile)**、**设计模式**、**需求分析**、**软件测试**与**项目管理**。
    
    软件工程旨在应对“软件危机”，确保开发出高质量、可维护、满足需求的软件产品，是现代大规模、协作化软件开发不可或缺的**指导框架**。
