# 面向对象程序设计

!!! abstract "课程简介"

    **C++** `C++ Programming Language`是一种支持**多范式**的静态类型编译语言。它在C语言的基础上，增加了**类**、**对象**、**继承**、**多态**等面向对象特性，以及**模板**、**异常处理**、**命名空间**等现代语言特性。
    
    **主要内容**：**类与对象**（封装、继承、多态）、**运算符重载**、**模板与泛型编程 (STL)**、**异常处理**以及现代 C++ 特性（RAII、智能指针）。
    
    C++在追求**高性能**与**资源控制**的同时，提供了更高层次的抽象能力，被广泛用于开发**游戏引擎开发**、**浏览器核心**、**金融高频交易系统**以及**大型桌面应用**等领域，是连接底层硬件控制与高级软件抽象的经典工业级语言。
