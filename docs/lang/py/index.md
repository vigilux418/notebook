# Python程序设计

!!! abstract "课程简介"

    **Python** `Python Programming Language`是一种解释型、高级、通用的编程语言，是现代科研与自动化的首选脚本语言。
    
    **主要内容**：动态类型系统、**装饰器**、**生成器**、**模块化开发**，以及丰富的第三方库应用（NumPy, Pandas, PyTorch）。
    
    凭借极高的开发效率，Python 统治了**数据科学**、**人工智能研究**、**后端自动化运维**及**网络爬虫**等领域，已经成为这些领域的**主流语言**。
