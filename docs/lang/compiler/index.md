# 编译原理

!!! abstract "课程简介"

    **编译原理** `Compilation Principles`， 也称编译技术。它系统地阐述了高级编程语言到低级目标代码（如汇编或机器码）的自动翻译过程。
    
    课程内容主要涉及前端，包括`tokenizer`、`parsing`，以及一些`Runtime`和`Memory`问题。
    
    编译原理不仅是**实现编程语言、解释器、静态分析工具的基础**，其核心思想也广泛应用于**自然语言处理**、**搜索引擎**、**程序分析**、**软件安全**及**领域特定语言**等领域，是计算机科学中理论与实践完美结合的典范。
