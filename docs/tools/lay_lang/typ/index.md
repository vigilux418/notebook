# Typst

!!! abstract "概要"

    **Typst** 是一门**现代化的、功能齐全**的现代**基于标记的排版系统**（Markup-based Typesetting System）。Typst由 **Rust** 编写，旨在解决 LaTeX 学习曲线陡峭、编译速度慢和宏系统复杂等历史痛点。Typst 引入了**增量编译**（Incremental Compilation）机制，实现了即写即看的实时预览体验，极大地优化了写作与排版的交互体验。
    
    在功能架构上，Typst 拥有强大的**脚本能力**和内置的**布局引擎**。它将文档结构与逻辑控制深度融合，通过类似于 Python 的**函数式编程语法**处理复杂的排版逻辑。其原生支持高质量的数学公式渲染（基于 Unicode-math）、现代字体特性（OpenType）以及灵活的绘图工具（如 Cei-ci 图表库），无需像 LaTeX 那样依赖冗余的宏包堆砌。
    
    目前，在学术排版领域，**Typst** 凭借其**高性能**、**高易用性**和**现代化工具链**，作为一种**LaTeX的现代化替代方案**正迅速获得关注。它不仅被视为**新一代学术写作工具**的有力竞争者，更在简历制作、课程论文、技术报告以及自动化文档生成等场景中展现出取代传统工具的巨大潜力。
