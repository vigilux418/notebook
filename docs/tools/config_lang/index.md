# 配置语言

!!! abstract "概要"

    **配置语言** `Configuration Language` 是用于定义程序运行参数、资源映射及系统元数据的声明式语言。**配置语言**并非用于描述算法或控制流，而是专注于以结构化、可读的方式描述**状态**与**配置**。
    
    **配置语言**部分涵盖当前主流的配置语言与数据序列化格式，将对比分析各种语言的**语法特性**及其**典型应用场景**等。
    
    配置语言是**基础设施即代码 (IaC)** 的基石。广泛应用于**应用程序配置**、**持续集成/持续部署**流程定义、**基础设施即代码**、**容器编排**及**API数据交换** 中。掌握高效的配置语言管理技巧，是构建可扩展、易维护的现代软件系统的先决条件。
