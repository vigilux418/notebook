# Vivado

!!! abstract "概要"

    **Vivado**是由AMD公司推出的集成设计环境，是**FPGA（现场可编程门阵列）**设计领域的行业标准工具套件。
    
    主要内容：涵盖较完整的Vivado使用流程，包括**项目创建与管理**、使用**Verilog**进行设计输入、**仿真**、**设计约束**、**实现**、**比特流生成与下载调试**，以及**IP封装与调用**。
    
    Vivado是**硬件描述语言 (HDL)**落地的标准平台。掌握Vivado是从事**数字系统设计**、**通信算法硬件加速**、**嵌入式系统开发** 以及**原型验证**等工作的必备实践技能。
