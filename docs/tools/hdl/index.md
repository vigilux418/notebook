# 硬件描述语言

!!! abstract "概要"

    **硬件描述语言** `Hardware Description Language`， 简称 **HDL**，是一种用于形式化描述数字电路和系统的专用编程语言，主要分为`Verilog`和`VHDL`。
    
    HDL不同于传统的用于编写**执行序列**的软件编程语言，而是用于描述电子电路的**结构**、**行为**和**时序**。HDL涵盖**并发执行策略**、**结构化建模**、**行为级建模**、**逻辑综合 (Synthesis)** 以及**仿真测试平台 (Testbench)** 的构建。
    
    HDL 是现代 **EDA (电子设计自动化)** 技术的基石，也是从事**数字集成电路**、**FPGA开发**、**ASIC设计**和**复杂数字系统建模**的基石。它使得工程师能够以高级抽象的方式设计、验证和实现复杂的硬件逻辑，是连接算法思想与物理电路的关键桥梁。
