# 标准Prompt

!!! abstract "概要"

    **标准 Prompt** `Standard Prompt Engineering` 是对生成式 AI 进行精确指令控制的结构化语言。
    
    其主要内容包括**Prompt的基本构成要素**、**通用模板结构**、**上下文管理技巧**等等，具体涉及**角色设定**、**任务描述**、**输出格式规范**、**思维链**引导等最基本的Prompt技巧。
    
    Prompt 工程是**提示词工程 (Prompt Engineering)** 的核心实践。通过标准化的指令，能够显著降低模型生成的**幻觉 (Hallucination)**，提升输出的确定性与专业度，是实现**人机协同开发**的必备技能。
