# 基础知识

!!! abstract "概要"

    **基础知识**`Foundational Knowledge`是本“AI指南”的理论介绍，为后续的实践应用提供必要的概念框架和理解背景。
    
    本部分并非对机器学习或深度学习理论的完整复现，而是对AI相关原理和机制的极简介绍，主要起到的是“扫盲”作用。内容通常包括大语言模型的基本工作原理、Transformer架构的直观理解，及`token`、`Agent`等关键术语的介绍。
    
    **基础知识**是从“调用 API”转向“算法理解”的关键。掌握这些基础知识，能够帮助开发者在遇到复杂任务时，从模型选型、数据预处理到超参数调优，都能具备最基本的理论支撑和直觉判断。
