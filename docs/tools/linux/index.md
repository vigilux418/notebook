# Linux

!!! abstract "概要"

    `Linux` 是基于开源协议发布的**类 Unix 操作系统内核**，1991年由 `Linus` 首次公开。它并非单一系统，而是衍生出Ubuntu、CentOS等众多发行版的核心。在技术生态中，它是服务器、嵌入式设备、超级计算机的**主流选择**————开源特性允许企业与个人自由修改代码，按需定制功能，大幅降低部署与维护成本，这构成了其在**基础软件领域**的**核心地位**。
    
    `Linux` 是理解**操作系统原理**的实践窗口，也是开发者高频使用的环境。学习 `Linux` 不仅能熟悉**命令行交互逻辑**，更能培养系统级问题排查思维，为后续深入系统编程、运维等领域打下必要基础。
