# 数字电路与逻辑

!!! abstract "课程简介"

    **数字电路与逻辑** `Digital Logic & Design` 是硬件系统的物理起点。
    
    主要内容：涉及**布尔代数**、**门电路**、**组合逻辑电路**（译码器、加法器）、**时序逻辑电路**（触发器、计数器、状态机）以及 **可编程逻辑器件 (FPGA/CPLD)**。
    
    **数字电路与逻辑**是理解计算机底层硬件如何表示和处理二进制信息（0/1）的关键，是后续学习**计算机组成**、**计算机体系结构**的先修课程，是连接软件理论与物理实现的桥梁。
