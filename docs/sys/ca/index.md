# 计算机体系结构

!!! abstract "课程简介"

    **计算机体系结构** `Computer Architecture`，简称 `CA`或 `Arch`。它是**计算机组成原理**的深化与扩展，主要从程序员（编译器/操作系统设计者）可见的视角，研究计算机系统的概念性结构与功能特性。
    
    主要内容：**指令集架构 (ISA)**、**流水线技术 (Pipelining)**、**指令级并行 (ILP)**、**存储层次结构优化 (Cache/Memory)** 以及**多处理器系统**。
    
    本课程旨在揭示如何通过硬件与软件界面的协同设计，在功耗、成本、性能等多重约束下，优化计算机系统的整体效能，是从事**高性能计算**、**处理器设计**、**编译优化**和**系统软件**开发的关键。
