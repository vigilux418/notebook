# 复变函数与积分变换

!!! abstract "课程简介"

    **复变函数**是高等学校工科类学生必须具备的工程数学知识，也是高等微积分的重要后继课程之一。它的理论与方法被广泛地应用于自然科学的许多领域，如电子工程、控制工程、理论物理与流体力学、弹性力学、热力学等，是专业理论研究和实际应用方面不可缺少的有力的数学工具。
    **拉普拉斯变换**作为复变函数在其他数学分支中的应用，同时也是工程技术中必不可少的重要数学内容。
